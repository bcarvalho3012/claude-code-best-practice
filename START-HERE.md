## What is Claude Code?

Claude Code is an AI assistant that runs in your terminal (the text window on your computer). You type instructions in plain English — like "fix this bug" or "add a login page" — and Claude reads your files, writes code, and runs commands for you.

## What is this repository?

This repo is a collection of tips, patterns, and working examples for getting the most out of Claude Code. Think of it as a **cookbook** — you don't need to read every recipe before you start cooking. Pick what's useful and skip what isn't.

## The 5 Things You Need to Know

### 1. Memory (CLAUDE.md)

A file you put in your project that gives Claude instructions it remembers every session. Like a sticky note your AI assistant reads every morning. Write things like "use Python 3.11" or "always write tests."

### 2. Commands

Shortcuts you type with a `/` — like `/weather-orchestrator`. They trigger pre-written instructions so you don't have to retype the same request every time. Think of them as speed dial buttons.

### 3. Skills

Bundles of knowledge that teach Claude how to do specific things — like creating a weather card or formatting code a certain way. Some activate automatically when relevant; others you trigger manually. Think of them as instruction manuals Claude can pick up and follow.

### 4. Sub-Agents

Independent copies of Claude that handle a specific subtask and report back. Like delegating a job to a coworker — they work separately, finish the task, and return with the result. Useful for complex work that benefits from focused attention.

### 5. Hooks

Small scripts that run automatically at specific moments — like a doorbell that rings when someone arrives. In this repo, sound effects play when Claude finishes a task or makes a commit.
